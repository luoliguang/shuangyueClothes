### 修改生成的目录标题
```javascript
export default {
  themeConfig: {
    outline: {
      level: [2, 3], // 显示的标题层级，默认是 [2, 3]
      label: '目录', // 自定义 "On This Page" 的标题
    },
  },
};
```
